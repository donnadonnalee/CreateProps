# CreateProps

Creator's Premium Utility Suite.

動画・画像・音声クリエイターのための便利ツール集です。

## Tools Included
- **Transparent PNG Studio**: 高度な色空間処理と勾配検知アルゴリズムを用いた、プロ仕様の半透明PNG作成・背景透過ツール。

## Privacy & Security
すべてのツールは純粋なスタティックWebアプリケーション（静的HTML/JS）です。**すべてのデータ処理はブラウザ上でローカルに行われます。**
アップロードした画像などの個人情報やファイルデータが外部サーバーに送信または保存されることは一切ありません。機密性の高いクリエイティブ素材も安心して処理していただけます。
